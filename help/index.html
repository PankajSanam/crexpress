Welcome

Current Version:

0.1.4 Beta

Crexpress is a mixed approach of a content management system and a framework. Crexpress is not just another effort to beat other frameworks or content management systems on web. This is not authors intention to insist users to use this framework rather than using most popular CMS like Wordpress, Joomla, Drupal and Most popular PHP Frameworks like CodeIgniter, CakePHP and Yii.

Crexpress is nothing but a simple approach that author found suitable for himself and he thinks that it would also be helpful for other developers and website owners. Author is trying to make this CMF as friendly as possible with lot of functionality. That doesn't mean it will be bloated by useless tools just like other frameworks and CMS.

I have found myself uncomfortable working with other users code and approaches. I simply wanted to develop something on my own and here we are. I wanted to include all good features from different CMS and frameworks that makes my general development work faster than any other CMS or framework would make.

Core Features

  1. Search Engine friendly URL's>
    • URLs produced by Crexpress mimic the structure of static website as they appear as html files which is mostly likeable by the search engines.
  2. Page and content management
    • Ability to add page templates and use them as master templates.
    • Add, edit, delete pages (There are some essential pages that cannot be deleted for site to be functional)
    • Enable / Disable pages and stop them showing on front end without deleting them.
    • Unlimited page hierarchies. A page can be used as a page category.
    • Rich text editor (ckeditor) for managing content
  3. Robust Htaccess
    • Added 301 redirection in htaccess
    • Added gzip and deflate compression in htaccess
    • Added htaccess to force all non-images files as images in vision/data folder
    • Added browser caching in htaccess
    • Added Options -Indexes in htaccess file to prevent directory listing and removed index.html from folders.
    • Added hotlink protection in htaccess.
    • Added upload file limits in htaccess.
    • Skip the download dialogue for specific file types in htaccess.
    • Prevent viewing specific file or multiple file types in htaccess.
    • Block IPs in htaccess.
    • Deny any request for static files (images, css, etc) if referrer is not local site or empty in htaccess.
    • Block visitors referred from specific domains in htaccess.
    • Added 404, 403 and 500 custom error pages in htaccess.
    • Addition in htaccess to prevent hacking via website URL.
  4. Admin Panel
    • Responsive and mobile friendly
    • Added gzip and deflate compression in htaccess
  5. User defined functions can be added in helper folder.
  6. Auto image resize, cropping and thumbnails.
  7. Added Gallery management
  8. Added User management
  9. Added Slider management
  10. Uses PDO extensions for better support and security
  11. Added general used methods in helper class to speed up work.
  12. Easy to use html tags without writing lots of code.
  13. Proper validation and sanitization.
  14. Better error reporting and debugging
  15. Lots of upload functionality for files with proper validation.
  16. Ability to add social icons dynamically.
  17. Added search functionality
  18. Better navigation with in-built functionality
  19. Autoloading of classes makes it faster by calling only those classes that are required.
  20. Ability to use meta tags for each page differently.
  21. Encryption functionality for locking and unlocking data with mcrypt.
  22. Better 404 error page handling.
  23. MVC approach by separating view from logic.
  24. Automatic log file creation for errors.
  25. Robust robot.txt file
  26. Message and enquiry manager
  27. Automatic html and xml sitemap creation feature
  28. Supports SMS API functionality.
  29. Supports email sending.
  30. Ability to select multiple menu positions for pages
  31. Added pagination
