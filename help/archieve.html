List of old features and work.

Core Features

  1. Search Engine friendly URL's
    • URLs produced by Retina CMF mimic the structure of static website as they appear as html files which is mostly likeable by the search engines.
  2. Pages Module
    • Ability to add new pages
    • Ability to edit any page
    • Ability to active/inactive any page
    • Ability to use pages as another pages category
  3. Image resizing and thumbnails
